Object Detection and Vibration Feedback Glove

Birmingham, United Kingdom

The Object Detection and Vibration Feedback Glove project represents an innovative integration of technology aimed at enhancing human-machine interaction for the blind and visually impaired. Developed as part of my academic pursuit in robotics, this project involved the creation of a real-time object detection system using depth and color cameras. By leveraging deep learning models and sophisticated computer vision techniques, the glove could accurately identify specific objects, ranging from everyday items to diverse environmental elements.
Moreover, this interactive system wasn't limited to mere detection. It was engineered to provide tactile feedback through integrated vibration motors, offering users real-time sensory information corresponding to the detected objects. Robust error handling and motor control mechanisms ensured not only the reliability of operation but also the safety protocols necessary for diverse usage scenarios.
The project's multidisciplinary approach amalgamated machine learning, computer vision, and robotics, exemplifying the fusion of theoretical knowledge and practical application. Through this endeavor, I honed my skills in system design, integration of advanced technologies, and the importance of user-centric solutions in the ever-evolving field of robotics.

Machine Learning - Friends Classifier

The Friends Image Classifier is a project that encompasses an array of technologies to recognize and categorize characters from the renowned TV series 'Friends'. Through the development process, I curated my own comprehensive database of character images, meticulously collected to train and refine our machine learning models. Haar Cascade Classifiers were used to search for facial features, cropping faces from the images and ensuring they had both eyes, enabling me to create a custom dataset.
I experimented with various machine learning models, including SVM, Random Forest, and Logistic Regression, to determine the most effective classification approach. After testing and analysis, Support Vector Machines (SVM) delivered the highest accuracy and precision in character recognition.
To bring this project to life, I established a Flask server hosted locally using PyCharm, enabling seamless access to our trained SVM model. This server provided a robust platform for users to upload images, initiating the classification process, and receiving instant results, showcasing the fusion of image processing, machine learning, and web development technologies in this innovative Friends Image Classifier.

Natural Language Processing - Smart Shopping ChatBot

The Chatbot Project exemplifies my expertise in Natural Language Processing (NLP) and neural network development. Designed for shopping-related inquiries, the chatbot interprets user inputs, such as greetings, shipping queries, payment concerns, expressions of gratitude, and farewells.
Leveraging NLTK for word tokenization and bag-of-words representation, I manually tagged these intents and trained a succinct two-layer neural network. The model effectively recognizes patterns in user input and generates contextually appropriate responses.
The project showcases my ability to create tailored conversational interfaces through a fusion of NLP and neural networks, highlighting a user-centric approach to technology.
